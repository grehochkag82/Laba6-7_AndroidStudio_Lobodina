# Tip Time Theme — Калькулятор чаевых

**Tip Time Theme** — это простое и удобное приложение для Android, разработанное в рамках лабораторной работы. Оно помогает быстро рассчитать сумму чаевых в зависимости от суммы счёта и выбранного процента. Приложение автоматически обновляет результат при вводе данных и поддерживает округление суммы.

## Цель работы

Целью данной лабораторной работы было изучение концепции **состояния (State)** в Android-приложениях и практическое применение **Jetpack Compose** для построения декларативного интерфейса. В ходе работы был реализован калькулятор чаевых, а также изучены основы написания **автоматизированных тестов** (локальных и UI) для проверки логики и пользовательского интерфейса.

## Функциональность

Приложение предоставляет следующий набор возможностей:

- **Ввод суммы счёта:** Поле для ввода с цифровой клавиатурой и иконкой денег.
- **Ввод процента чаевых:** Поле для ввода с цифровой клавиатурой и иконкой процента.
- **Автоматический пересчёт:** Итоговая сумма чаевых обновляется в реальном времени по мере ввода данных.
- **Округление суммы:** Переключатель (Switch), позволяющий округлить сумму чаевых до целого числа в большую сторону.
- **Удобная навигация:** Клавиша "Next" на клавиатуре для перехода к следующему полю и "Done" для её закрытия.
- **Адаптивный интерфейс:** Поддержка портретной и альбомной ориентации с вертикальной прокруткой.

## Технологии и инструменты

В проекте были использованы следующие технологии и библиотеки:

- **Язык:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Дизайн-система:** Material Design 3
- **Архитектурные подходы:** State Hoisting (подъём состояния), Stateless Composables
- **Сборка:** Gradle (KTS)
- **Тестирование:**
    - JUnit4 (для локальных unit-тестов)
    - Compose UI Testing (для инструментальных тестов интерфейса)
